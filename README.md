# Excel Assistant v1.0.0

A smart local Excel file management tool with natural language interface.

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python excel_assistant_enhanced_gui.py`
3. Type "show help" to see available commands

## Features
- Smart auto-complete with arrow key navigation
- Natural language command processing
- Built-in searchable help system
- 100% local - no internet required
